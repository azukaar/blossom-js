L-SET

The component is a made to set a context's value. the most simple case is to change an existing context.

this.ctx

You can notice if you inspect the DOM that l-set keep track of the set, by setting an additional prop `has_set_foo`, in order not to repeat the setting. This is useful to initialise context. You might be unfamiliar with the system of context, you might want to read this to learn more.
