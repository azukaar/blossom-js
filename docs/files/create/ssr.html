SERVER SIDE RENDERING

Using the package blossom-js-server-side, you can easily render a Blossom page on Node environment. This is useful to provide server-side-rendering, as well as for unit testing. Warning : if performing any Ajax in your app, always use the fetch() methods to do it. 

      npm install blossom-js-server-side.
/** * @jest-environment node */ import 'blossom-js-server-side'; import 'blossom-js'; const template = ` Hello World Goodbye `; const rendered = BlossomRender(template); console.log(rendered.innerText) // Hello World
